# 🌤️ Консольное приложение для получения прогноза погоды по API

Простой и эффективный Python-скрипт для получения текущих погодных данных в любом городе мира с использованием сервиса **OpenWeatherMap API**. 

Программа отправляет запрос к API, обрабатывает полученный JSON-ответ и выводит структурированную, легко читаемую информацию о погоде прямо в консоль, а также сохраняет полный лог ответа в файл.

---

## ✨ Основные возможности

* **Актуальные данные:** Получение температуры, влажности, давления, скорости ветра и текстового описания погоды в реальном времени.
* **Локализация:** Описание погоды выводится на русском языке, а температура — в градусах Цельсия.
* **Надежность:** Реализована комплексная обработка ошибок (сетевые сбои, неверный API-ключ, отсутствие подключения, ошибки парсинга JSON).
* **Экспорт данных:** Автоматическое сохранение полного ответа от API в файл `weather_data.json` для последующего анализа.

---

## 🛠️ Стек технологий

* **Язык:** Python 3.x
* **Библиотеки:** * `requests` — для отправки HTTP-запросов.
  * `json` — для работы с форматом данных JSON.
